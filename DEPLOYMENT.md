# Deployment Instructions

## Deploy to Streamlit Community Cloud

1. **Visit Streamlit Cloud:**
   Go to https://share.streamlit.io/

2. **Sign in:**
   Sign in with your GitHub account (anmacmillan)

3. **Create New App:**
   - Click "New app"
   - Repository: `anmacmillan/case-success-predictor`
   - Branch: `main`
   - Main file path: `app.py`
   - App URL: Choose `case-success-predictor` (becomes case-success-predictor.streamlit.app)

4. **Deploy:**
   Click "Deploy!" and wait for the app to build (2-3 minutes)

5. **Verify:**
   Once deployed, visit https://case-success-predictor.streamlit.app to test

## Features to Test
